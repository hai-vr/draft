Layers 22 to 31 included are for world creators

Layers 22 to 31 are free to be used by world creators. The name of the layers are always the same in-game, regardless of the Unity project settings.

Layer 22 is often used for PostProcessing, but you don't have to.

Layer names in-Editor and in VRChat may be different

When working with Udon, the layer names of the project are ignored, and will be different in-Editor than in VRChat.

You should use the layer index instead.

Use bit shifting to represent layer masks

If you need a raycast to hit 3 specific layers, let's take for example Environment, Pickup and Walkthrough.

The layers are respectively 11, 13, 17.

The layer mask that represents them is:

int hitsEnvironmentPickupAndWalkthrough = 1 << 11 | 1 << 13 | 1 << 17;

There are other ways to define this, but this is a simple format that keeps it reasonably readable.

If you put this as the layer mask for a raycast, it will only hit those layers.

Use the complement operator to represent everything except some layers

If you need a raycast to everything except 2 specific layers, let's take for example Environment and Pickup.

The layers are respectively 11, 13.

The layer mask that represents them is:

int hitsEverythingExceptEnvironmentAndPickup = ~(1 << 11 | 1 << 13);

If you put this as the layer mask for a raycast, it will hit everything except those layers.
